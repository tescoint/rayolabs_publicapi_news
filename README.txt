The RayoLabs Content King plugin (developed by publicapi.com.ng) is powerful, lightweight and easy to use Atom/RSS aggregation and content curation plugin for WordPress. The Content King plugin has one very unique feature which allows you to import full text articles.

== Description ==

This plugin connects your wordpress website to https://publicapi.org.ng and provides a webhook endpoint where new records can be pushed directly to your wordpress website.

To get started, headover to https://publicapi.org.ng and create an account if you don't have one, next you create a new token on publicapi, now this token is what you will insert in the plugin admin page and that's all.
